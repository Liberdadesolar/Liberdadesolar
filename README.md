## Liberdade Solar

Energia solar para condominios — sem investimento, com economia garantida.

Instalamos micro-redes solares em condominios residenciais com **financiamento 100% proprio**. O morador economiza **~20% na conta de luz** desde o primeiro dia, sem pagar nada pela instalacao.

### Como funciona

```
Visita tecnica gratuita
        |
Instalacao sem custo para o condominio
        |
Economia de ~20% na conta de luz
        |
Monitoramento 24/7 pelo app
```

### Tecnologia

- **Assistente IA** com Claude (Anthropic) — atende duvidas de moradores, sindicos e parceiros em tempo real
- **Streaming SSE** — respostas em tempo real
- **PWA** — funciona como app no celular
- **Deploy automatico** via Vercel + GitHub
